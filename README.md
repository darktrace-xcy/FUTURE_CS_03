# API Security Risk Analysis

Overview

This project demonstrates a basic API Security Risk Analysis conducted on the public demo API **JSONPlaceholder** using **Postman**.

The assessment focused on:

* API endpoint testing
* Header inspection
* Authentication review
* Data exposure analysis
* Security risk identification



Tools Used

* Postman
* Browser DevTools
* JSONPlaceholder API



API Tested

[JSONPlaceholder API](https://jsonplaceholder.typicode.com?utm_source=chatgpt.com)



Key Security Findings

* Unauthenticated API access
* Excessive data exposure
* Broken Object Level Authorization (BOLA)
* Weak authentication for POST/DELETE requests
* Missing visible rate limiting


Recommendations

* Implement JWT/OAuth authentication
* Enforce authorization checks
* Apply rate limiting
* Reduce unnecessary data exposure
* Validate API requests properly

 Project Includes

* API Security Report
* Postman testing screenshots
* Risk assessment
* Security recommendations
